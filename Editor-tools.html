Prefab sorter

Due to a NDA i am not allowed to show any code. However i can eplain what this tool does.

This is a tool that VROwl requested me to make. They can import this tool with an Unity package and use to it sort prefabs and all dependencies of the prefabs. 

How the tool works

There are two sort options, prefabs and materials. The prefabs will be sorted in the given 
main-catagoty -> sub-gatagorie. Here it will create 4 folders prefabs, materials, textures and objects. It will also create a scene in this folder for the showcase. 

After pressing the sort button. The prefab and dependencies will be sorted and spoawned in a scene for a preview.

This can also be done for materials. Only the destination folder will be Assets/Materials instead of Assets/main-catagory/sub-gatagory.

These are the main catagories, with the creation of a new sub folder. Done by pressing the + button in the re-orderable-list.
Added cube 1 to Example -> Sub Folder as shown in the list. The list now shows all the prefabs in this sub folder.
These are the settings for the sorter. When the "run" buttons is pressed and the spawn in settings are on.
All the prefabs in the sub folder will be instantiate in the scene that is located in the same sub folder.

As the options show you can also choose to spawn in the materials, shown as a sphere.

The left object is a controller button with a red material in case you where wondering...
